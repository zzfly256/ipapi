API 使用方法

  • 使用 POST/GET 方式，向 ip.zzfly.net 发起请求
  • 所需字段：ip
  • 返回结果： 
    {
    "ip": "119.29.29.29",
    "area": "广东省深圳市",
    "isp": "烟台帝思普网络科技有限公司公共DNS服务器(BGP节点)",
    "ip_segment": "119.29.29.29"
}
  • 调用示例 PHP： 
    <?php var_dump(file_get_contents(json_decode("http://ip.zzfly.net?ip=需要查询的IP")));?>
  • 调用示例 Shell： 
    curl -d "ip=需要查询的IP" "http://ip.zzfly.net"
